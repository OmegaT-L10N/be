Што такое OmegaT?
===============

OmegaT — гэта свабодная і шматплятформенная праграма на базе памяці перакладаў
з магчымасьцямі падбору недакладных супадзеньняў, падтрымкай памяці перакладаў,
пошукам паводле ключавых словаў, глясароў і выкарыстаньня перакладаў
у абноўленых праектах.

Умовы для ўсталёўкі OmegaT
===================================

OmegaT патрабуе ўсталёўкі ў сыстэме Java Runtime Environment
(асяродзьдзя выкананьня Java) вэрсіі 1.8 ці пазьнейшай.

Раім карыстацца OmegaT у выглядзе пакетаў з JRE, тады не спатрэбіцца
яго шукаць, атрымліваць і ўсталёўваць.



Усталёўка OmegaT (Windows)
===========================

Запусьцеце праграму ўсталёўкі



Усталёўка OmegaT (Mac)
===========================

Распакуйце архіў OmegaT.zip, каб атрымаць каталёг з файлам
дакумэнтацыі і праграмным файлам OmegaT. Перамясьцеце каталёг у адпаведнае месца,
напрыклад, у каталёг праграмаў.



Усталёўка OmegaT (Linux)
===========================

Пакладзеце архіў у які заўгодна прыдатны каталёг
і распакуйце яго. Пасьля гэтага можна запусьціць OmegaT.

Аднак вы можаце скарыстацца з акуратнейшага і зручнейшага
для карыстальніка спосабу ўсталёўкі з дапамогай
сцэнара ўсталёўкі (linux-install.sh). Для выкарыстаньня
гэтага сцэнара адчынеце вакно тэрмінала (кансолі), перайдзеце ў каталёг,
дзе знаходзяцца файлы OmegaT.jar і linux-install.sh (сцэнар усталёўкі),
і выканайце сцэнар камандай ./linux-install.sh.



Усталёўка OmegaT (Solaris, FreeBSD, ...)
=========================================

Пакладзеце архіў у які заўгодна прыдатны каталёг
і распакуйце яго. Пасьля гэтага можна запусьціць OmegaT.



Выкарыстаньне Java Web Start для ўсталёўкі OmegaT (усе плятформы)
===========================================================

Калі Java ўжо ўсталяваная, бягучую вэрсію OmegaT
можна ўсталяваць праз Java Web Start.

Для гэтага трэба загрузіць наступны файл і затым выканаць яго:

   https://omegat.sourceforge.io/webstart/OmegaT.jnlp

Ён усталюе на ваш кампутар належнае асяродзьдзе і самую праграму
пры першым запуску. Для далейшых выклікаў злучэньне з Інтэрнэтам неабавязковае.



Запуск OmegaT (Windows)
===========================

Калі ў працэсе ўсталёўкі вы стварылі цэтлік на працоўным стале,
двойчы клікнеце па ім.

Праграма ўсталёўкі можа стварыць цэтлікі ў мэню пуску,
на працоўным стале і на панэлі хуткага выкліку. Вы таксама можаце ўручную перацягнуць
файл OmegaT.exe ў мэню пуску, на працоўны стол або панэль хуткага запуску,
каб зрабіць там адпаведнае лучыва.

Калі вы бачыце файл OmegaT, але не OmegaT.exe праз
мэнэджар файлаў (Windows Explorer), трэба зьмяніць налады так,
каб паказваліся пашырэньні назваў файлаў.



Запуск OmegaT (Mac)
===========================

Двойчы клікнеце па праграме OmegaT.

Можна перацягнуць праграму OmegaT у свой док або на панэль акна Finder,
каб атрымаць магчымасьць запуску зь якога заўгодна месца. Праграму таксама можна будзе запусьціць
праз пошукавае поле Spotlight.



Запуск OmegaT (Linux)
===========================

Калі быў выкарыстаны сцэнар усталёўкі, для запуску OmegaT трэба націснуць:

  Alt+F2

а затым увесьці:

  omegat

Для зручнейшага запуску OmegaT можна карыстацца сцэнаром Kaptain
з архіва (omegat.kaptn). Каб выкарыстаць гэты сцэнар, неабходна найперш усталяваць
Kaptain. Пасьля гэтага можна запусьціць сцэнар запуску Kaptain праз

  Alt+F2

а затым увесьці:

  omegat.kaptn



Запуск OmegaT з каманднага радка (усі сыстэмы)
====================================================

Каманда запуску OmegaT наступная:

cd <каталёг, дзе знаходзіцца файл OmegaT.jar>

<назва і шлях да выканальнага файла Java> -jar OmegaT.jar

(Выканальны файл Java — гэта файл java ў Linux і java.exe ў Windows.
Калі Java ўсталяваная на сыстэмным узроўні і прапісаная ў камандным шляху,
поўны шлях не патрэбны.)
